# Planning & Narrative Documents

> ⚠️ **DESCRIPTIVE ONLY** - Documents in this folder describe intent, strategy, or progress.
> They are not generated by CI or automation. Do not treat as verification evidence.

---

## What Belongs Here

- Progress summaries and "what we did" writeups
- Strategy reasoning documents
- Roadmaps and "what's next" planning
- Phase summaries that describe work rather than verify it
- Discussion notes and decision records

## What Does NOT Belong Here

- Verification checklists → `docs/verification/`
- Runbooks with executable commands → `docs/guides/`
- Architecture contracts → `docs/architecture/`
- Security policies → `docs/security/`
- Competitive analysis → `docs/competitive/`

---

*All documents in this folder are historical artifacts for context, not execution evidence.*
