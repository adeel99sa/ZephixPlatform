# Guides

This folder contains operational and development guides for the Zephix platform.

## Contents

- **LOCAL_DEVELOPMENT_FULL_STACK.md** - PostgreSQL + NestJS + Vite on macOS (Docker Postgres, migrations, daily workflow)
- **ENGINEERING_PLAYBOOK.md** - Development standards, code style, PR process
- **OPERATIONS_RUNBOOK.md** - Production operations, monitoring, incident response
- **DEPLOYMENT_GUIDE.md** - Railway deployment, CI/CD, environment configuration
- **QA_TESTING_GUIDE.md** - Testing procedures, smoke tests, UAT checklists
- **STAGING_PUSH_TRACKER.md** - Running list of changes queued before pushing to staging (update as you ship)

## Default Entry Point for New Work

- **IMPLEMENTATION_CHANGE_TEMPLATE.md** - Required pre-implementation intake, lane ownership, gate checks, proof expectation, and rollback criteria

## Guidelines

- Keep guides actionable and up-to-date
- Include step-by-step procedures where applicable
